# Vitaminka Tap Rush

A fast-paced browser game developed for **Vitaminka**. Players compete by tapping as quickly as possible to achieve the highest score within a limited time. The project combines an engaging user interface with real-time score storage using Firebase.

## Project Overview

Vitaminka Tap Rush is a web-based game created to promote user engagement through a simple yet addictive tapping challenge. The game tracks player scores and stores data using Firebase, allowing for a more interactive experience.

## Features

- Fast-paced tap/click gameplay
- Score tracking system
- Countdown timer
- Real-time data storage with Firebase
- Responsive user interface
- Dynamic game feedback and results

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Configure Firebase credentials if required.
4. Open `index.html` in a web browser.
